# Furniture Room Designer

A modern starter frontend for a furniture room design website built with Next.js, React, TypeScript, and Tailwind CSS.

## Terminal Commands

```bash
mkdir furniture-room-designer
cd furniture-room-designer
npm install
npm run dev
```

Open `http://localhost:3000` to view the site.

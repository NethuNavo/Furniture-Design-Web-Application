# Furniture Design Web Application


This project is a web application for furniture visualization and room design.

## Overview
This project is a web-based Furniture Room Designer that allows users to design and visualize furniture arrangements inside a room.

Users can define room dimensions, customize appearance, and place furniture items to create their own interior layout. The system provides both 2D and 3D visualization to improve understanding of space and design.

## Features
- User authentication (signup, login)
- Room customization (size, shape, colors)
- 2D room planner with drag-and-drop furniture
- 3D room visualization
- Furniture customization (color, size, placement)
- Save and manage designs (with mock/local data)
- Furniture browsing (shop section)

## Technologies Used
- Next.js
- React
- Tailwind CSS
- TypeScript
- Prisma (for backend structure)
- Three.js (for 3D visualization)

## System Modules
- Authentication Module
- Room Designer Module (2D + 3D)
- Saved Designs Module
- Shop Module
- Admin Panel (basic structure)

## Project Goal
The goal of this project is to improve user experience in interior design planning by applying Human-Computer Interaction (HCI) principles such as usability, efficiency, and learnability.

## Notes
- Some features use mock data due to partial backend implementation.
- The main focus of the project is frontend design and user interaction.
